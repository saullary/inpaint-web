# [Inpaint-web](https://github.com/lxfater/inpaint-web)

A free and open-source inpainting tool powered by webgpu and wasm on the browser.

基于 Webgpu 技术和 wasm 技术的免费开源 inpainting 工具, 纯浏览器端实现。
